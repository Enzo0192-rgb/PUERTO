# PUERTO DE SANTA — Centro de Control Portuario

Versión ajustada a la zona real indicada por el usuario:

- Todas las operaciones se concentran en la zona del puerto de carga.
- Camiones más pequeños y ubicados dentro de la playa operativa.
- Buques/barcazas posicionados sobre el agua.
- Mantiene escenarios, KPIs, alertas, costos e inversión.
