# PUERTO DE SANTA FE — Centro de Control Portuario V2

Dashboard operativo interactivo para presentación del proyecto.

## Estructura

```text
index.html
README.md
assets/
css/
  style.css
  dashboard.css
  map.css
js/
  app.js
  map.js
  simulator.js
  charts.js
  scenarios.js
  animations.js
data/
  puerto.json
  escenarios.json
  costos.json
```

## Cómo publicarlo en GitHub Pages

Subir todos los archivos y carpetas a la raíz del repositorio `PUERTO`.

La estructura debe verse así:

```text
PUERTO/
  index.html
  README.md
  assets/
  css/
  js/
  data/
```

Luego abrir:

```text
https://enzo0192-rgb.github.io/PUERTO/index.html
```

## Funciones

- Mapa satelital interactivo.
- Camiones ubicados en la playa del puerto.
- Barcazas ubicadas sobre el río.
- Zonas operativas editables.
- Capas activables/desactivables.
- Escenarios Hoy, +2, +5 y +10 años.
- Simulador de inversiones.
- Estimador de costos.
- KPIs dinámicos.
- Alertas operativas.
