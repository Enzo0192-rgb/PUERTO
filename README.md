# Puerto de Santa Fe — Centro de Control Portuario

Versión interactiva para presentación.

## Estructura

```text
index.html
README.md
assets/
  dashboard_puerto_santa_fe.png
```

## Uso

Subir los tres elementos a GitHub Pages y abrir:

```text
https://enzo0192-rgb.github.io/PUERTO/index.html
```

## Funciones

- Escenarios: Hoy, +2, +5 y +10 años.
- Simulador de inversiones.
- Estimador de costos por espera.
- Panel de barcazas en río.
- KPIs dinámicos.

La imagen principal representa el centro de control y el panel derecho permite manipular alternativas operativas.
