## Descripción

Esta versión usa una imagen satelital fija como base visual del tablero para representar con precisión:

- camiones dentro de la playa del puerto;
- barcazas ubicadas sobre el río;
- zonas operativas concentradas en el predio;
- panel de KPIs;
- escenario operativo;
- distribución de carga;
- estimador visual de operación.

Esta base visual permite avanzar luego hacia una versión interactiva con simulación de costos, turnos, colas, inversiones y escenarios futuros.
